Raízes do Futuro - Atividade 3

Este projeto foi desenvolvido para a Atividade 3 da faculdade, com foco em JavaScript avançado, transformando a interface estática em uma aplicação web dinâmica e interativa.

Tecnologias Utilizadas

<span style="color: #1e90ff; font-weight:bold;">HTML5</span> – estrutura das páginas

<span style="color: #17c964; font-weight:bold;">CSS3</span> – layout, cores, responsividade e animações

<span style="color: #ff7f50; font-weight:bold;">JavaScript</span> – manipulação do DOM, eventos, validação de formulários e armazenamento local

Funcionalidades Desenvolvidas

Sistema de Single Page Application básico, permitindo navegar entre Início, Projetos e Cadastro

Validação de campos obrigatórios no formulário de cadastro

Máscaras automáticas para CPF, Telefone e CEP

Armazenamento de dados no localStorage, garantindo que o usuário possa retornar e ver os campos preenchidos

Exibição de mensagens de feedback com toast para indicar sucesso ou erro no envio

Manutenção completa do layout original, incluindo centralização do formulário, botões de ação e cards dos projetos

Estrutura do Projeto

index.html – página inicial com apresentação da ONG

projetos.html – página mostrando os projetos ativos

cadastro.html – formulário para cadastro de voluntários

assets/css/style.css – estilos, cores e animações

assets/js/script.js – funcionalidades em JavaScript

assets/imagens/ – logos, ícones e imagens dos projetos
